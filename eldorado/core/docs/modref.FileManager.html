Название

Файловый менеджер — Управление файлами на диске

О модуле

ТипАдминистративный
Можно создать копийТолько одну
Присутствует по умолчаниюДа
Можно удалятьНет

Файловый менеджер позволяет управлять файлами и каталогами на сервере, как Проводник в Windows. С его помощью можно:

  • загружать файлы на сервер;

  • скачивать файлы с сервера;

  • создавать, переименовывать и удалять папки;

  • копировать, перемещать, удалять группы файлов;

  • создавать и распаковывать архивы tar.gz, tar.bz2, zip и rar (если он установлен в системе);

Файловый менеджер используется в следующих модулях:

  • Файловый архив - управление файлами,

  • ???? - выбор файла баннера.

Для остальных модулей ручное вмешательство в файловую систему обычно не требуется.

Доступные действия

Перейти в

С помощью этого выпадающего списка можно быстро переместиться в нужный каталог.

Действия → К началу

Перейти в корневой каталог.

Действия → Создать директорию

Создать новую папку в текущей.

Действия → Загрузить файлы

Загрузить файлы на сервер.

Действия → Создать архив

Создать архив в текущей директории, поместив в него нужные файлы из нее.

Действия → Копировать/переместить/удалить группу файлов

Копировать или переместить в другую директорию или удалить указанные файлы из текущей директории. В появившейся форме необходимо выбрать из выпадающего списка нужное действие, целевую директорию и выбрать нужные файлы.

Настройки → Настройки

Кол-во одновременно загружаемых файлов

Сколько файлов можно будет загрузить на сервер за один раз.

Кол-во иконок в ряду

Сколько значков файлов отображать в ряду на странице.

Кол-во иконок в ряду (всплывающее окно)

Сколько значков файлов отображать в ряду на странице, когда файловый менеджер открыт в отдельном окне для выбора файла на какой-то другой странице.

Максимальный размер миниатюрки (px)

Размер миниатюр, генерируемых для графических файлов.
