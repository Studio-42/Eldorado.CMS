Установка

1. Загрузка программы установки на сервер

Сначала вам нужно загрузить файл installer.php в веб-каталог на вашем хостинге, воспользуйтесь для этого FTP-клиентом или панелью управления вашим хостингом. Положите этот файл в тот каталог, в котором должен быть расположен ваш сайт.

2. Установка прав доступа к каталогу

Для установки вам необходимо разрешить веб-серверу запись в каталог, в который вы положили файл программы установки, для этого нужно изменить права доступа к этому каталогу. Это можно сделать в большинстве FTP-клиентов или в файловом менеджере в панели управления веб-хостингом. Права доступа обычно представлены в виде девяти флажков, установите их все для данного каталога. Если же ваш FTP-клиент требует введения цифрового кода прав доступа, внесите 777, этот код соответствует установке всех флажков.

3. Запуск программы установки

Перейдите в вашем браузере по адресу: http://ваш_домен/installer.php, должна открыться первая страничка программы установки.

3.1. Выбор языка установки

На первом этапе необходимо выбрать язык, на котором будут выводится сообщения при установке ELDORADO.CMS. Список языков зависит от присутствующих в системе языковых пакетов.

3.2. Лицензионное соглашение

На втором шаге вам необходимо прочитать и принять условия лицензионного соглашения. Отказ от принятия условий лицензии прекращает процесс установки.

3.3. Тип установки

На третьем шаге нужно выбрать тип установки из следующих трех вариантов:

Установка одиночного сайта

Это самый простой вариант - установка одного отдельного сайта, если вы не уверены, какой вариант вам нужен, выбирайте этот.

Мультисайтовая установка с установкой ядра системы

Такой тип установки следует выбирать, если вы планируете применить совместное использование ядра системы несколькими сайтами и собираетесь установить первый сайт из группы использующих общее ядро.

Мультисайтовая установка. Ядро системы уже установлено

Этот тип установкинужно выбирать для всех сайтов использующих общее ядро кроме первого, установленного по второму варианту.

Замечание

Даже, если вы планируете устанавливать множество сайтов, вы можете тем не менее выбрать установку одиночного сайта, это не повлияет на работоспособность системы.

3.5. Установка шаблонов дизайна сайта

На этом шаге вы можете выбрать из списка один из стандартных вариантов оформления сайта либо загрузить свои шаблоны (если для вас был специально разработан и сверстан дизайн), указав файл, содержащий шаблоны.

3.6. Пароль администратора сайта

На этом шаге нужно ввести пароль администратора сайта и его мастер-пароль. Это просто два разных пароля для одного администратора, он сможет входить используя любой из них.

Замечание

Настоятельно рекомендуется придумывать сложный пароль, как минимум из 10 символов, среди которых должны быть и цифры, и буквы в верхнем и нижнем регистре.

3.7. Завершение установки

После нажатия на кнопку "Завершить" вы будет перенаправлены на только что созданный сайт.

4. Настройка веб-сервера

Если при попытке перейти на какую-либо страницу нового сайта или войти с логином и паролем, сервер выдает ошибку 404 "Страница не найдена", то вам необходимо в настройках веб-сервера настроить для папки, в которую установлена система, возможность управления модулем mod_rewrite. Обратитесь для этого в службу поддержки вашего хостинга.

Далее вам необходимо произвести первоначальную настройку сайта.
