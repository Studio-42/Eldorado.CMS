==========================
   Установка
==========================

Скрипт поставляется с инсталятором, который позволяет определить настройки сервера и сравнить с необходимыми требованиями. Для начала создайте отдельную папку (или виртуальный сервер) под скрипт;

Перепишите в нее содержимое ZIP архива в виде которого поставляется скрипт;

Пользователи ОС UNIX/Linux: необходимо сделать chmod 777 (во многих FTP клиентах эта функция доступна. Например в CuteFTP или FAR) для папки includes. (Это дает права скрипту записывать в эту папку данные необходимые для правильной работы);

Если Вы хотите использовать новую базу данных под скрипт, то Вы должны создать ее. Я рекомендую использовать программу phpMyAdmin, она очень удобна для администрирования MySQL через веб-интерфейс. Инструкции по ее использованию можно найти у Вашего хостера, либо в Интернете. Если Ваш хостер не предоставляет этого средства или Вы устанавливаете скрипт на локальный компьютер, то можно использовать такие программы как например MySQL-Front;

Запустите установочный скрипт index.php из папки _setup. Для этого наберите в браузере http://сайт/путь/_setup/index.php. Произойдет проверка параметров и в случае положительного результата можно переходить к заполнению форму с настройками подключения к базе данных;

Далее Вы будите видеть на экране ход установки скрипта. При возникновении каких либо ошибок вы увидите сообщение о них. ВНИМАНИЕ: Не прерывайте установку до получения сообщения о полной установки скрипта;

Удалите с сайта папку _setup и все находящиеся в ней файлы. Так как появляется возможность переписать ренее введенные Вами данные, то это необходимо в целях безопасности скрипта;

Обязательно закрывайте доступ к папке includes, чтобы никто не смог узнать Ваши пароли подключения к базе. Если Вы пользуетесь HTTP-сервером Apache, Вы можете сделать это с помощью .htaccess файла.

==========================
   Настройка
==========================

Сложной настройки скрипта не требуется. После успешной установки войдите в скрипт как администратор набрав в браузере http://сайт/путь/admin/ и введите используемый по умолчанию логин и пароль admin. Здесь Вы можете поменять настройки доступа к административной части скрипта, указать настройки необходимые для его работы. Первое что рекомендуется сделать — изменить пароль для доступа к скрипту. Обращаю внимание, что логин admin поменять нельзя! Одна из полезностей данного раздела — это информация о выходе новых версий скрипта.

На страницы своего сайта с которых хотите получать статистические данные нужно вставить HTML код (скрипт должен быть установлен на этом сайте) который будет выдан Вам на странице администрирования.

==========================
   Отчеты
==========================

Для получения статистических данных наберите в браузере http://сайт/путь/index.php. В этом окне Вы можете выбирать вид отчета, его представление в разных графических формах, экспортирование отчетов на E-Mail, в PDF, Word и EXCEL файлы.

Для изменения внешнего вида скрипта под дизайн своего сайта предусмотрены HTML файлы. С помощью них Вы можете управлять положением графиков и таблиц на странице.

==========================
   Страны, регионы, города и организации
==========================

Если Вы хотите получать эти виды отчетов, то можете использовать внутреннюю GeoIP-базу скрипта.

Для получения статистических данных Вы должны скачать в сайта файлы (они доступны в разделе загрузки):

GeoIPCity.zip - Страны, регионы и города. 
GeoIPOrg.zip - Организации 
После этого распаковать их в папку /includes/geoip/

   Приятной работы!
