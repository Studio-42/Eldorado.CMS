Название

Резервные копии сайта — Создание резервных копий и восстановление сайта из них.

О модуле

Параметры
ТипАдминистративный
Можно создать копийТолько одну
Присутствует по умолчаниюНет
Можно удалятьДа

С помощью этого модуля создается раздел резервного копирования сайта. В этом разделе можно создавать резервные копии, управлять ими и восстанавливать из них сайт. По умолчанию страница этого типа присутствует в ___контрольном центре___.

Замечание

Настоятельно рекомендуется делать резервные копии сайта настолько часто, насколько это возможно без ущерба для производительности сервера.

Общие действия для раздела

Действия → Создать резервную копию

Создать новую резервную копию текущего состояния сайта.

Действия → Загрузить файл резервной копии

Загрузить резервную копию сайта с локального компьютера на сервер.

Настройки → Настройки

Редактирование параметров резервного копирования.

Действия над резервными копиями

Восстановить из резервной копии

Восстановление сайта из резервной копии.

Внимание

Внимание! Все текущие данные сайта будут утеряны при восстановлении, сайт полностью вернется в состояние на момент сохранения восстанавливаемой резервной копии. Рекомендуется сделать дополнительную резервную копию перед восстановлением.

Удалить

Удаление выбранной резервной копии.

Параметры резервного копирования

Дисковая квота для резеврных копий (Mb)

Ограничение места на диске, отведенного для хранения резервных копий сайта.

Создавать резервные копии автоматически

Создавать резервные копии автоматически с указанным интервалом.
