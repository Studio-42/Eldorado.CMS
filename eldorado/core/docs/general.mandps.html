Модули и плагины

Основная функциональность ELDORADO.CMS распределена по модулям. Каждый модуль дает возможность создавать страницы определенного типа, например, фотоальбом, новости, каталог ссылок. Количество модулей в системе постепенно увеличивается, улучшается и их функциональность. Выбор типа страницы осуществляется при создании новой страницы.

В настоящее время система содержит следующие модули:

  • Альбомы изображений - с помощью этого модуля вы можете создавать фотогалереи.

  • Голосования - позволяет создавать опросы.

  • Интернет-магазин - виртуальный магазин.

  • Карта сайта - отображает структуру сайта.

  • Каталог вакансий - здесь можно разместить вакансии вашей компании, удобно рассортированные по группам.

  • Каталог документов - позволяет создать подборку документов (статей) по разной тематике, рассортированных по группам и подгруппам, виртуальную библиотеку и т.п.

  • Каталог ссылок - упорядоченное собрание ссылок.

  • Каталог тех. товаров - модуль специально разработан для более удобного представления технических товаров - от мобильных телефонов до промышленных станков, легко позволяет присваивать объектам различные модели и технические характеристики, при включении соответствующей опции легко превращается в интернет-магазин.

  • Каталог товаров

  • Контейнер - позволяет объединить несколько страниц различных типов в одну группу. На странице контейнера отображаются ссылки на вложенные страницы и краткое описание.

  • Новости - лента новостей.

  • Обновление системы - позволяет обновить ELDORADO.CMS до последней версии.

  • Одиночная страница - страница с текстом или графикой.

  • Отправка e-mail - почтовая форма стандартного вида, позволяет отправлять вам сообщения посетителям сайта.

  • Почтовые формы - конструктор почтовых форм, с помощью этого модуля вы можете создавать формы для связи произвольного вида, что удобно, например, для принятия заказов по e-mail и т.п.

  • Редактор дизайна - позволяет вручную редактировать все html-шаблоны и css-файлы сайта.

  • Расписание событий

  • Словарь терминов - позволяет создать глоссарий.

  • Статистика - информация о посещаемости сайта.

  • Файловый архив - позволяет создать каталог файлов (например, mp3-архив).

  • Файловый менеджер - управление файлами.

  • Часто Задаваемые Вопросы - с помощью этого модуля легко создать FAQ (часто задаваемые вопросы), посетители сайта смогут отправлять вам свои вопросы прямо с сайта.

Также в системе есть дополнительные модули (плагины) - небольшие информационные блоки, которые могут быть размещены на любой странице сайта.

  • Информационные блоки - позволяет делать короткие объявления, анонсы и т.п.

  • Заголовки новостей - позволяет выводить заголовки и краткие анонсы новостей.

  • Голосования - размещает опросы на любой странице.

  • Случайная картинка - изображения из галереи изображений на любой странице сайта.
