Название

Импорт и экспорт — Совместное использование данных разными разделами

Описание

Импорт и экспорт данных позволяют организовать передачу данных между разделами сайта.

Параметры импорта

Разрешить импорт данных

Производить или нет импорт.

Адрес источника данных (URL)

URL страницы с которой будет осуществляться импорт

Параметр импорта

Строка, использующаяся как критерий отбора.

Ключ импорта

Пароль для доступа к данным.

Период автоматического импорта данных (дни)

Как часто нужно импортировать данные из источника.

Дата последнего импорта

Этот параметр нельзя изменить, он устанавливается автоматически при каждом импорте данных.

Параметры экспорта

Разрешить экспорт данных

Разрешить или запретить другим разделам импортировать данные из этого раздела.

Ключ экспорта

Секретный пароль, который должен передать другой раздел, чтобы получить доступ к данным.

Критерий отбора

Для каждой единицы данных (такой как новость в колонке новостей или событие в расписании событий) можно указать "Параметр экспорта" - произвольную строку, по которой потом можно будет отобрать только нужные данные, указав в параметрах импорта импортирующей страницы такой же "Параметр импорта". При этом на импортирующую страницу будут помещены только те единицы данных, у которых "Параметр экспорта" совпадает с указанным "Параметром импорта".
