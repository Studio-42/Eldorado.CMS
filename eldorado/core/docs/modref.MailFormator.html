Название

Почтовые формы — Конструктор почтовых форм. Создание форм для опросов, анкет, обратной связи и т.п.

О модуле

ТипПубличный
Можно создать копийСколько угодно
Присутствует по умолчаниюНет
Можно удалятьДа

С помощью этого модуля на сайте создаются почтовые формы. Почтовая форма представляет собой страничку с несколькими полями для заполнения и кнопкой отправки. Данные отправленной формы придут вам в виде обычного сообщения электронной почты на указанный адрес. Такие формы используются для различных опросов, анкет и т.п.

Подсказка

Для создания форм обратной связи используйте модуль Отправка E-mail.

Форма составляется из элементов, которые выстраиваются друг за другом вертикально. Модуль позволяет добавлять, изменять и удалять такие элементы.

Доступные действия

Действия → Создать новый элемент

Добавление нового элемента почтовой формы. При создании элемента могут быть указаны перечисленные ниже параметры. Некоторые из них нужны только для определенных типов элементов, для остальных типов такие параметры скрываются.

Настройки → Настройки

Редактирование параметров формы.

Параметры элемента

Позиция элемента

Положение элемента в форме. Верхний элемент имеет индекс 1, следующий - 2, и т.д.

Тип элемента

Один из перечисленных ниже.

Название элемента

Подпись к элементу.

Значение элемента по умолчанию

Начальное значение элемента, отображаемой при открытии странички почтовой формы.

Варианты значений

Только для выпадающего списка, группы выключателей и группы кнопок-переключателей (см. Типы элементов).

Тип значения

Для проверки введенных посетителем данных и выдачи сообщения при ошибке ввода. Для всех типов кроме комментария и заголовка группы элементов (см. Типы элементов).

Сообщение об ошибке

Сообщение, котрое будет выдаваться пользователю при неверно введенных данных.

Параметры формы

Заголовок формы

Текст подтверждения

Сообщение, отображаемое после отправки формы.

Тема сообщения

Заголовок сообщения электронной почты, которое будет приходить с сайта. Если ничего не указывать, будут приходить сообщения с темой "Сообщение с сайта".

Разрешить выбор получателя

Разрешить или нет посетителям выбирать адресата сообщения из общего списка адресов.

Получатели

Получатели отправленного сообщения.

Типы элементов

Комментарий

Некоторый пояснительный текст отображаемый в форме.

Заголовок группы элементов

Заголовок - выделенный текст с центральным выравниванием.

Текстовое поле

Простое поле для ввода строки текста. При создании нужно указать название, начальное значение и его тип. Тип используется для проверки введенного посетителем значения на допустимость. также можно ввести сообщение об ошибке, которое будет выдаваться пользователю при вводе некорректного значения.

Текстовая область

То же, что и текстовое поле, но предназначено для многострочного текста.

Выпадающий список

Поле с набором предопределенных значений. Кроме подписи и начального значения нужно также определить список возможных значений и обязательность заполнения.

Группа выключателей

Группа флажков, каждый из которых может быть установлен или снят независимо от остальных.

Группа кнопок-переключателей

Переключатель, состоящий из нескольких кнопок, из которых включена может быть только одна.
