Пред. Справочник по общим элементам и механизмам След.

Название

Параметры соединения с базой данных — Хост, сокет, база данных, пользователь и пароль.

Описание

Эти параметры необходимы для установки соединения с базой данных, они указываются при установке системы, а также для указания централизованного хранилища учетной информации при организации портала. Эти параметры вам должен сообщить ваш администратор базы данных.

Хост

Имя или адрес компьютера, на котором установлена база данных. Как правило она установлена на том же компьютере, что и сам сайт, в этом случае нужно указать localhost.

Сокет

Обычно указывать его не требуется, но если нужно, то ваш администратор базы данных сообщит, что нужно указать.

База данных

Имя базы данных.

Пользователь

Пользователь базы данных.

Пароль

Пароль пользователя базы данных.
