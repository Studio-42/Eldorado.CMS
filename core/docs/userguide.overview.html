Что такое ELDORADO.CMS

ELDORADO.CMS - это система управления сайтом, программный продукт разработанный для создания и успешного управления интернет-ресурсом любого уровня, начиная от промо-сайта и заканчивая он-лайн магазином или крупным торговым порталом. Уникальный, интуитивно понятный интерфейс редактора сайта позволяет с легкостью работать с любой информацией на сайте, не обладая при этом никакими специальными навыками.

Система управления Eldorado CMS разработана таким образом, что позволяет менять функциональность и добавлять новые возможности вашему сайту в любое время, а простота интерфейса в сочетании с широкими возможностями настройки способна удовлетворить потребности как простых пользователей так и опытных специалистов.

  • простота установки и настройки системы,

  • уникальная концепция интерфейса администратора,

  • модульная структура CMS,

  • многосайтовость,

  • гибкая система распределения прав доступа пользователей,

  • оптимизация и быстродействие системы,

  • управление настройками любого компонента системы управления.

Система состоит из ядра и набора модулей.

Ядро системы - базовая часть программного кода, обеспечивающая полноценное функционирование системы управления сайтом. Ядро создает программную среду, для работы основных и вспомогательных модулей, контролирует соблюдение прав доступа и выполняет другие необходимые функции. При разработке комплекса программных решений, объединенных общим понятием ядра, учитывалась перспектива дальнейших разработок и меняющихся с течением времени, требований к функциональности Интернет-ресурсов.

Основные функции ядра Eldorado.CMS:

  • создание необходимого программного контекста для функционирования системы управления сайтом,

  • корректное отображение данных с учетом визуального дизайна сайта,

  • управление правами доступа, авторизация и регистрация пользователей,

  • управление структурой сайта,

  • средства редактирования информации,

  • поиск по сайту,

  • обеспечение почтовым сервисом,

  • управление файлами,

  • обеспечение работы пользовательского интерфейса,

  • поддержка разных языковых версий сайта.
