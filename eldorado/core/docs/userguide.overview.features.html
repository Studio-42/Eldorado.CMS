Особенности системы

Модульная структура

Основная функциональность ELDORADO.CMS распределена по модулям. Каждый модуль дает возможность создавать страницы определенного типа, например, фотоальбом, новости, каталог ссылок.

Полный список доступных модулей приведен в справочнике по модулям.

Непосредственное редактирование (WYSIWYG)

В ELDORADO.CMS применена концепция непосредственного редактирования, это значит, что в системе отсутствует отдельный интерфейс для редактирования страниц, вместо этого на каждой странице сайта, администратор имеет возможность переключиться в режим редактирования и непосредственно изменить ее содержимое.

Разделение оформления и содержимого

В ELDORADO.CMS визуальное оформление сайта полностью отделено от смыслового наполнения и программного кода с помощью шаблонов. Для изменения дизайна достаточно внести изменения в шаблоны, которые представляют собой обычные HTML-страницы.

Многоязычность

Система поддерживает простое переключение языка всех встроенных названий, надписей и сообщений. В настоящий момент доступны языковые пакеты для двух языков:

  • русский,

  • английский.

Многосайтовость

ELDORADO.CMS позволяет создавать как одиночные сайты, так и порталы, состоящие из нескольких связанных сайтов. Для этого применяется два механизма:

  • централизованное хранение учетной информации нескольких сайтов,.

  • передача данных между разделами сайтов.

Совместное использование ядра системы несколькими сайтами

ELDORADO.CMS позволяет при развертывании нескольких сайтов на одном веб-хостинге, установить только одну копию программного ядра и использовать ее для всех устанавливаемых сайтов, не важно, будут ли какие-либо из этих сайтов объединены в портал или все сайты будут отдельными.

Такой вариант установки системы позволяет упростить обновление системы.
