Создание структуры сайта

После установки и начальной настройки необходимо создать разделы сайта.

Разделы создаются на странице модуля Управление навигацией сайта ( Контрольный центр → Управление навигацией ). Список всех доступных типов разделов приводится в справочнике по модулям.

Сразу после создания, разделы пусты, для наполнения перейдите в нужный раздел с помощью главного меню сайта и отредактируйте его непосредственно.

Кроме создания страниц сайта с помощью модулей, имеется возможность добавлять на страницы различные динамические элементы с помощью плагинов.
