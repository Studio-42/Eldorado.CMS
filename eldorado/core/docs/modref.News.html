Этот модуль позволяет создать на сайте раздел новостей. Новость представляет собой небольшую статью в которой описывается какое-либо событие. Новости объединяются в колонку новостей с сортировкой по дате.

Новости из данного раздела могут отображаться в боковой панели или в другой части сайта на нужных страницах, для этого применяется плагин Заголовки новостей.

Общие действия для колонки

Действия → Добавить новость

Создать новую новость, указав ее параметры.

Настройки → Настройки

Редактировать параметры ленты новостей.

Настройки → Настройка импорта данных

Редактирование параметров импорта.

Настройки → Настройка экспорта данных

Редактирование параметров экспорта.
