Название

Просмотрщик изображений — Механизмы отображения больших изображений

Описание

В настройках модуля фотогалереи и плагина случайного изображения для отображения больших картинок можно выбрать один из следующих двух механизмов:

LightBox

Изображение плавно открывается поверх текущей страницы, которая при этом затемняется. При щелчке по правому краю изображения открывается следующая картинка, а по левому - предыдущая, внизу расположена кнопка закрытия просмотра. Для того, чтобы этот способ работал, браузер посетителя должен выполнять JavaScript.

Новое окно

Полноразмерное изображение открывается в новом окне браузера.
