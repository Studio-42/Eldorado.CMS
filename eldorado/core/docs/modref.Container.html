Название

Контейнер — Создание вложенных страниц

О модуле

ТипПользовательский
Можно создать копийСколько угодно
Присутствует по умолчаниюНет
Можно удалятьДа

Модуль позволяет объединять несколько страниц в один раздел, создавая таким образом древовидную структуру. Если рассматривать аналогию с организацией папок и файлов на диске, то контейнер это папка, которая может содержать в себе файлы (страницы сайта). Пример контейнера - страница Контрольного Центра.

Доступные действия

Настройки → Настройки.

Позволяет выбрать режим работы контейнера - показать список вложенных страниц или перенаправлять на первую вложенную страницу.
