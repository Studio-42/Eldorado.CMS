Название

Каталог вакансий — Упорядоченное собрание объявлений о вакансиях

О модуле

ТипПубличный
Можно создать копийСколько угодно
Присутствует по умолчаниюНет
Можно удалятьДа

Этот модуль создает на сайте раздел, содержащий набор вакансий, сгруппированных по категориям, как файлы сгруппированы по папкам на диске компьютера.

Общие действия для каталога

Действия → Новая категория

Создание новой категории с указанием ее свойств.

Действия → Новая вакансия

Добавление новой вакансии с указанием её свойств.

Действия → Сортировать документы в категории

Задание явного порядка сортировки документов в данной категории. Документы с ненулевым приоритетом помещаются в начало и выстраиваются по его возрастанию, документы с нулевым приоритетом сортируются по полю, указанному в параметрах каталога.

Действия → Удалить группу документов

Выбрать ненужные документы из текущего каталога и удалить их.

Настройки → Настройки

Редактирование общих параметров каталога.

Настройки → Настройка навигации каталога

Редактирование параметров дополнительного меню каталога.

Включает отображение описания текущей категории, задаваемое в свойствах категории.

Параметры формы для отправки резюме:

Список получателей

Выбрать получателей из списка имеющихся на сайте адресов e-mail (задаются в ***контрольном центре*** - Контрольный центр → Настройки сайта → Адреса e-mail).

Форма для отправки резюме

Использовать форму по умолчантю - предлагать посетителю стандартную форму для отправки резюме, создать в конструкторе формы - создать форму поизвольного вида. После того, как выбрана эта опция, в меню "Настройки" появляется дополнительный пункт - "Конструктор формы", запускающий модуль "Почтовые формы".

Разрешить отсылать резюме во вложенном файле

При включенной опции посетителю разрешается приложить свое резюме в виде текстового файла.

Тескт подтверждения

Сообщение, отображающееся после успешной отправки формы.
