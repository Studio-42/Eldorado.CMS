Каталог ссылок — Коллекция ссылок на другие сайты

О модуле

ТипПубличный
Можно создать копийСколько угодно
Присутствует по умолчаниюНет
Можно удалятьДа

С помощью этого модуля можно создать на сайте каталог ссылок на другие интернет-ресурсы.

Общие действия для каталога

Действия → Новая категория

Создание новой категории с указанием ее свойств.

Действия → Новая ссылка

Добавление новой ссылки с указанием ее свойств.

Действия → Сортировать ссылки в категории

Задание явного порядка сортировки ссылок в данной категории. Ссылки с ненулевым приоритетом помещаются в начало и выстраиваются по его возрастанию, ссылки с нулевым приоритетом сортируются по полю, указанному в параметрах каталога.

Настройки → Настройки

Редактирование общих параметров каталога.

Действия над категориями

Редактировать

Изменение свойств категории.

Вверх

Переместить категорию на одну позицию вверх в списке.

Вниз

Переместить категорию на одну позицию вниз в списке.

Удалить

Удалить пустую категорию.

Действия над ссылками

Редактировать

Изменить свойства ссылки.

Вверх

Переместить ссылку на одну позицию вверх в списке.

Вниз

Переместить ссылку на одну позицию вниз в списке.

Удалить

Удалить ссылку.

Параметры каталога

Сколько уровней каталога отображать за раз

Сколько уровней вложенности категорий каталога отображать на одной странице.

Вид списка категорий

В одну или в две колонки.

Вид списка документов

Отображать список ссылок в одну или в две колонки.

Сортировать

Способ сортировки ссылок, для которых не установлен явный приоритет сортировки. Возможные значения:

  • по названию,

  • по дате публикации,

  • по приоритету.

Кол-во документов на странице

Количество ссылок, отображаемых на одной странице.

Показывать описание текущей категории

Показывать или нет описание категории, задаваемое в свойствах категории.

Свойства категории

Родительская категория

Категория в которую будет помещена данная.

Название

Название категории.

Описание

Краткое описание категории, отображаемое в списке категорий под ее названием (форматированный текст).

Свойства ссылки

Родительская категория

Категория, в которую будет помещена ссылка.

Название

Название сайта.

Содержание

Описание сайта (форматированный текст).

URL

URL сайта.

Дата публикации

День, когда была опубликована ссылка.
