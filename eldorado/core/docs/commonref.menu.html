Название

Дополнительное меню раздела — Дополнительное меню в разделах со сложной структурой

Описание

Для некоторых разделов, которые могут иметь сложную структуру из многих вложенных подразделов, имеется возможность отображать дополнительное меню, облегчающее навигацию по разделу. Кроме того, такое меню (если для него настроено отображение на всех страницах сайта) позволяет посетителю быстро перейти в этот раздел с любой страницы сайта.
