Централизованное хранение учетной информации нескольких сайтов

Это один из двух механизмов, предназначенных для создания портала на основе ELDORADO.CMS. Суть его заключается в том, что все сайты протала используют одну и туже базу данных клиентов, в то время как остальные данные хранятся в собственной базе для каждого сайта. Это позволяет посетителям, зарегистрировавшись на одном из сайтов портала, автоматически получить учетную запись и на всех остальных сайтах.

Чтобы задействовать этот механизм, нужно выбрать один из сайтов портала, который будет считаться основным, для остальных сайтов в настройках контроля доступа сайта нужно установить удаленный тип авторизации и указать параметры базы данных основного сайта.
