Название

Расписание событий — Временной график каких-либо мероприятий

О модуле

ТипПубличный
Можно создать копийСколько угодно
Присутствует по умолчаниюНет
Можно удалятьДа

Данный модуль позволяет создать на сайте раздел, где будет представлено расписание каких-либо событий, например каких-нибудь мероприятий. На основной странице раздела представлен собственно список событий с возможностью посмотреть подробную информацию по каждому событию, которая представляет собой одну обычную страницу.

Общие действия

Действия → Новое событие

Добавить событие, указав его параметры.

Настройки → Настройки

Редактировать параметры расписания.

Настройки → Настройка импорта данных

Редактирование параметров импорта.

Настройки → Настройка экспорта данных

Редактирование параметров экспорта.

Действия над событием

Редактировать

Изменить параметры события.

Удалить

Удаление событие.

Параметры расписания

Показать дату как

Показывать дату со временем или без.

Показать место проведения

Отображать или нет данные о месте события.

Показать прошедшие события

Показывать или нет события, которые уже произошли в общем списке.

Параметры события

Название

Заголовок события.

Начало

Дата начала события. Если в параметрах расписания установлено отображение времени, то здесь также можно указать время.

Окончание

Дата окончания события. Если в параметрах расписания установлено отображение времени, то здесь также можно указать время.

Место проведения

Место проведения события. Это поле доступно только, если в параметрах расписания установлено отображение места проведения.

Анонс

Краткое описание события. отображаемое в основном списке событий (форматированный текст).

Описание

Полное описание события, открывающееся при щелчке по ссылке "Читать дальше" в списке событий (форматированный текст).
