Этот модуль создает на сайте раздел, содержащий набор обычных страниц, сгруппированных по категориям, как файлы сгруппированы по папкам на диске компьютера.

Общие действия для каталога

Действия → Новая категория

Создание новой категории с указанием ее свойств.

Действия → Новый документ

Добавление нового документа с указанием его свойств.

Действия → Сортировать документы в категории

Задание явного порядка сортировки документов в данной категории. Документы с ненулевым приоритетом помещаются в начало и выстраиваются по его возрастанию, документы с нулевым приоритетом сортируются по полю, указанному в параметрах каталога.

Действия → Удалить группу документов

Выбрать ненужные документы из текущего каталога и удалить их.

Настройки → Настройки

Редактирование общих параметров каталога.

Настройки → Настройка навигации каталога

Редактирование параметров дополнительного меню каталога.

Настройки → Настройка групп связанных объектов

Задание имен для групп связанных объектов (каталогов, имеющихся на сайте). Благодаря этой функции можно со страницы одного объекта ссылаться на объекты других каталогов. Ссылки на объекты других каталогов будут отображаться под заголовками, задаваемыми этой опцией.

Настройки → Настройка импорта данных

Настроить импорт данных.

Настройки → Настройка экспорта данных

Настроить экспорт данных.
