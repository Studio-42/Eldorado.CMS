Передача данных между разделами сайтов

Передача данных между разделами - это один из двух механизмов организации портала на основе ELDORADO.CMS. Данные могут передаваться как между двумя разделами одного сайта, так и между двумя разными сайтами.

Данные могут передаваться между такими разделами:

  • Новости,

  • Расписание событий.

  • Интернет-магазин,

  • Каталог (витрина) товаров,

  • Файловый архив,

В передаче данных принимают участие два раздела, один из них экспортирует данные, а другой - импортирует. Каждая единица данных в экспортирующем разделе помечается определенным критерием отбора, по которому импортирующий раздел может отобрать только часть данных. Это позволяет создать один общий раздел и экспортировать данные в несколько тематических.

Подробно о настройке передачи данных читайте в разделе Импорт и экспорт справочника по общим механизмам.
