Первоначальная настройка

После того, как система была установлена, требуется установить некоторые основные параметры сайта. Это делается на странице модуля настроек сайта в контрольном центре. Чтобы контрольный центр стал доступен, вам нужно войти в систему, указав учетное имя root и пароль, который вы указали во время установки системы.

  1. В подразделе общих настроек следует указать:

    • название сайта,

    • владельца сайта,

    • контактную информацию,

    • номера телефонов.

  2. В подразделе адресов email нужно указать как минимум один адрес электронной почты, на который будет отправляться корреспонденция с сайта.

После выполнения первоначальной настройки вы получаете полностью работоспособный сайт, имеющий всего две публичные страницы:

  • главная страница, которая представляет собой простую одиночную страницу;

  • страница обратной связи.

После первоначальной настройки вам необходимо создать структуру сайта.
