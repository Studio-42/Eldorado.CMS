В этом модуле сосредоточены глобальные настройки влияющие на сайт в целом. Переход в режим редактирования настроек осуществляется через верхнее меню (Настройки → Настройки. Модуль состоит из следующих подразделов:

  • общие настройки,

  • настройки разметки,

  • адреса email,

  • код счетчиков,

  • контроль доступа.

Общие настройки

Контрольный центр → Настройки сайта → Общие настройки

Название сайта

Отображается в титулах всех страниц сайта, если в настройках страницы не задано альтернативное название.

Владелец сайта

Информация о лице или фирме, которой принадлежит сайт. Отображается на каждой странице сайта, как правило мелким шрифтом внизу страницы.

Контактная информация

Адрес или другие реквизиты для контакта с владельцем сайта. Отображается на каждой странице сайта, как правило мелким шрифтом внизу страницы.

Номера телефонов

Контактные телефоны владельца сайта. Отображается на каждой странице сайта, как правило мелким шрифтом внизу страницы.

Использовать кэш

Кэш - это механизм позволяющий снизить нагрузку на сервер, на котором расположен сайт, путем записи на диск сгенерированных системой страниц и последующей выдачи сохраненных копий вместо постоянной повторной их генерации. Этот механизм следует применять при высокой посещаемости сайта.

Включить сжатие страниц

Позволяет отправлять пользователю страницы в сжатом виде, что позволяет уменьшить трафик и увеличить скорость загрузки. Как правило, эту опцию всегда полезно включить.

Показывать отладочные сообщения

Эти сообщения помогают разработчикам искать и исправлять ошибки в системе. Не следует без необходимости включать эту опцию, т.к. это может нарушить безопасность сайта.

Показывать время работы скрипта

Отображать время, затраченное системой на генерацию текущей страницы. Как правило это интересно только разработчикам, эта информация может помочь определит причину медленной работы сайта.

Язык интерфейса и кодировка

Позволяет выбрать языковой пакет из установленных в системе.

Валюта

Валюта, используемая в модулях Каталог (витрина) товаров и Интернет-магазин.

Настройки разметки

Контрольный центр → Настройки сайта → Настройки разметки

В этом подразделе можно настроить расположение на странице стандартных визуальных элементов сайта.

Адреса email

Контрольный центр → Настройки сайта → Адреса email

В этом подразделе можно составить список адресов электронной почты, из которых потом можно будет выбирать адрес для отправки сообщений из почтовых форм.

Действия → Новый e-mail

Добавление нового адреса. На появившейся странице необходимо ввести метку - название почтового ящика и адрес e-mail. Если выбрана необязательная опция "использовать по умолчанию", то данный адрес будет автоматически выбираться в качестве адреса получателя при создании новой почтовой формы.

Настройки → Настройки

В настройках этого подраздела можно указать какие сообщения нужно записывать в лог-файл и каким способом отправлять почту, с помощью внутренних механизмов иди через внешний SMTP-сервер (это может понадобится, если один из путей на вашем хостинге по какой-то причине не работает).

Для уже существующих адресов электронной почты доступны следующие действия:

Редактировать

Изменение данных, введенных при создании

Удалить

Удалить электронный адрес из списка

Код счетчиков

Контрольный центр → Настройки сайта → Код счетчиков

Здесь можно разместить HTML-код счетчиков посещаемости, после этого счетчики будут отображаться на каждой странице сайта (как правило, внизу). Для этого нужно выбрать пункт меню Настройки → Настройки, в появившуюся форму ввести необходимый HTML-код и нажать кнопку "Отправить".

Контроль доступа

Контрольный центр → Настройки сайта → Контроль доступа

Таймаут сессии

Через какое время не-активности пользователя его сеанс будет автоматически завершен.

Разрешить регистрацию новых пользователей

Включите эту опцию, если вы хотите сделать на сайте раздел с доступом только для зарегистрированных посетителей. Кроме этого потребуется настроить ограничения видимости в основных параметрах страницы общих настроек модуля.

Группа для новых пользователей

Группа, в которую будут автоматически включаться новые пользователи зарегистрировавшиеся самостоятельно или созданные администратором.

Блокировать уведомления по e-mail о регистрации пользователя

Не высылать уведомления пользователю при регистрации на сайте.

Блокировать уведомления по e-mail о смене пароля

Не высылать уведомления пользователю после смены его пароля.

Блокировать уведомления о регистрации нового пользователя на адрес по умолчанию

Не высылать уведомления администратору при регистрации нового пользователя.

Тип авторизации
Локальная

Учетная информация хранится в той же базе данных, что и остальные данные для данного сайта, эта схема применяется для обычного одиночного сайта.

Удаленная

Установка этого варианта задействует механизм централизованного хранения учетной информации для организации портала из нескольких сайтов. При выборе этого варианта открываются дополнительные поля параметров базы данных авторизации.
