Название

Голосования — Активные голосования

О плагине

Этот плагин отображает в правой или левой колонке активные голосования из указанного раздела голосований, причем если посетитель уже принимал участие в голосовании, то отображаются текущие результаты.

Настройки

Использовать этот источник

Для каждой существующей на сайте страницы голосований в настройках плагина появляется блок параметров, идентичный для каждой страницы. Установка данного флажка подключает указанную страницу голосования в качестве источника данных для плагина и, соответственно, голосования с этой страницы отображаются в указанной колонке на указанных страницах сайта.
