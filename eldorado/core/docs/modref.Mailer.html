Данный модуль позволяет создать на сайте страницу обратной связи. С такой страницы каждый посетитель может отправить сообщение администрации сайта, при этом email-адрес администрации явно нигде не отображается. Это позволяет избежать попадания ваших email-адресов в списки распространителей спама, куда бы они непременно попали, размести вы их в открытом виде на страницах сайта. Кроме того такая форма позволяет отправлять сообщения пользователям, не имеющими в данный момент возможности или желания отправлять его с помощью почтового клиента.

Параметры

Заголовок формы

Надпись крупными буквами вверху формы.

Разрешить выбор получателя

Позволить посетителю выбирать получателя сообщения из общего списка адресов.

Поле "Тема сообщения" обязательно

Требовать ли обязательного заполнения поля темы сообщения.

Метка поля "Тема сообщения"

Как будет называться поле темы (иногда его бывает необходимо назвать как-то иначе.

Тема сообщения по умолчанию

Текст. автоматически вставляющийся в поле темы при открытии формы.

Метка поля "Текст сообщения"

Как будет называться большое поле для ввода текста сообщения.

Верхний комментарий

Некоторый пояснительный текст, отображаемый в верхней части формы.

Нижний комментарий

Некоторый пояснительный текст, отображаемый в нижней части формы.

Текст подтверждения

Сообщение выдаваемое пользователю после успешной отправки формы, например: "Спасибо, Ваше сообщение отправлено, в самое ближайшее время мы свяжемся с Вами".
