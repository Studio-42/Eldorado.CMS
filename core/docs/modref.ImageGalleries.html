Альбомы изображений — Галерея фотографий или других изображений

С помощью этого модуля можно создавать и управлять коллекциями изображений. Это могут быть фотоальбомы, галереи или любые другие собрания изображений.

Вся галерея делится на альбомы, каждый из которых содержит коллекцию изображений.

Имеется плагин, который может отображать случайную картинку из галереи на каждой странице сайта.

Действия для галереи

Действия → Новый альбом

Создание нового альбома. Для альбома нужно указать название и можно добавить какой-нибудь комментарий.

Действия → Новое изображение

Добавление новой картинки в один из альбомов. При этом нужно заполнить свойства изображения.

Действия → Принудительная сортировка альбомов

Задание порядка отображения альбомов в галереи.

Действия → Импорт изображения из архива

Файл должен иметь одно из следующих расширений: tar.gz, tar.bz2, zip

Действия → Поиск и исправление ошибок

Эта функция позволяет восстановить галерею в случаях, когда в результате ручного вмешательства (например с помощью файлового менеджера) какие-то из файлов изображений были удалены, переименованы, либо были добавлены новые файлы. При нажатии на эту иконку производится два действия:

  1. Из галереи удаляются все не отображаемые картинки, файлы которых были физически удалены с диска.

  2. В галереи создается новый альбом под названием "Lost and found" и в него помещаются все найденные "бесхозные" картинки, которые не были загружены на сайт не через интерфейс галереи и лежат на диске. но ни в одном альбоме не отображаются.

Настройки → Настройки

Редактирование общих настроек галереи.

Общие настройки галереи

Показывать полноразмерное изображение, используя

Выбор способа отображения картинок в полный размер при щелчке по ним в просмотре альбома.

Максимальный размер миниатюрок (px)

Размер уменьшенных изображений выводимых при просмотре галереи или альбома. Оригинальная картинка будет пропорционально сжата, чтобы уместиться в указанную квадратную область.

Использовать уникальные имена файлов изображений

Генерировать сложное уникальное имя для каждого загруженного изображения.

Сортировать альбомы

Порядок, в котором выводятся альбомы в галереи. Возможные варианты: по названию, по дате публикации и по индексу сортировки.

Сортировать изображения

Порядок сортировки изображений в альбомах. Возможные варианты: по названию, по дате публикации и по индексу сортировки.

Количество миниатюрок в списке альбомов

Количество уменьшенных изображений отображаемых в каждом из альбомов в их списке на главной странице галереи.

Показывать дату создания альбома

Отображать или нет дату альбома в списке альбомов.

Показывать количество изображений в альбомов

Отображать или нет количество изображений в каждом из альбомов в списке альбомов.

Количество миниатюрок на странице

Максимальное количество уменьшенных изображений на каждой странице при просмотре альбома.

Количество миниатюрок в ряду

Количество уменьшенных изображений в каждом горизонтальном ряду при просмотре альбома.

Показывать размер оригинального изображения

Отображать исходный размер загруженной на сервер картинки в подписи к ней.

Показывать дату публикации изображения

Отображать дату загрузки изображения в его подписи.

Показывать имя файла изображения

Отображать имя файла изображения в его подписи.

Показывать размер файла изображения

Указывать размер файла изображения в его подписи.

Действия над альбомами

Редактировать

Изменить свойства альбома - его название и комментарий.

Удалить все изображения из альбома

Очистить альбом.

Удалить

Удалить альбом, можно удалить только пустой альбом.

Действия → Принудительная сортировка изображений в текущем альбоме

Указание собственного порядка сортировки для изображений в данном альбоме. Эта опция появляется только после выбора одного из альбомов в галереи.

Размер изображений

В этом выпадающем списке можно выбрать размер изображения при его индивидуальном просмотре.

Действия над изображениями

Редактировать

Изменить свойства изображения - альбом, в который оно входит, название, и комментарий, также можно заменить само изображение, загрузив новое.

Удалить

Удалить изображение.
