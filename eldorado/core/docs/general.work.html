Принципы работы с системой

В ELDORADO.CMS применена концепция непосредственного редактирования, т.е. в системе отсутствует отдельный интерфейс для редактирования страниц, вместо этого, на каждой странице сайта администратор имеет возможность переключиться в режим редактирования и непосредственно изменить её содержимое.

Переключение в режим редактирования осуществляется двумя способами. Либо нажатием на соответствующую иконку, которая есть в уголке любого элемента, подлежащего редактированию, либо через верхнее дополнительное меню (Действия-Редактировать).

Верхнее дополнительное меню отображается только для авторизованных пользователей. Пункты меню различаются в зависимости от модуля текущей страницы а также от прав пользователя. С помощью него можно переключаться в режим редактирования страницы, осуществлять настройки модуля страницы и дополнительных модулей. Через верхнее меню осуществляется управление содержанием модулей, состоящих из разных элементов (фотоальбом, новости, каталоги, интернет-магазин, и т.д.).

Добавление новых страниц и редактирование структуры сайта осуществляется через Контрольный центр (Контрольный центр → Навигация).
