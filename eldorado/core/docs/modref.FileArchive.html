Название

Файловый архив — Каталог файлов

О модуле

ТипПубличный
Можно создать копийСколько угодно
Присутствует по умолчаниюНет
Можно удалятьДа

Этот модуль позволяет организовать на сайте файловый архив - упорядоченное хранилище файлов, где посетитель сможет легко найти нужный файл и скачать его. Файлы сгруппированы по категориям, каждая категория может содержать в себе файлы и подкатегории, аналогично папкам в файловой системе.

Общие действия для архива

Действия → Новая категория

Создание новой категории с указанием ее свойств.

Действия → Новый файл

Добавление нового файла с указанием его свойств.

Настройки → Настройки

Редактирование параметров архива.

Настройки → Настройка навигации каталога

Редактирование параметров дополнительного меню архива.

Действия над категориями

Редактировать

Изменение свойств категории.

Вверх

Переместить категорию на одну позицию вверх в списке.

Вниз

Переместить категорию на одну позицию вниз в списке.

Удалить

Удалить пустую категорию.

Действия над файлами

Редактировать

Изменить свойства файла.

Удалить

Удалить файл.

Параметры архива

Сколько уровней каталога отображать за раз

Сколько уровней вложенности категорий архива отображать на одной странице.

Вид списка категорий

В одну или в две колонки

Вид списка документов

В одну или в две колонки

Сортировать

Способ сортировки файлов. Возможные значения:

  • по названию,

  • по имени файла,

  • по времени изменения файла,

  • по количеству скачиваний.

Количество документов на странице

Количество файлов, отображаемых на одной странице.

Показывать описание текущей категории

Отображать или нет описание категории, задаваемое в свойствах категории.

Показывать дату изменения файла

Время последнего изменения файла.

Показывать счетчик скачиваний

Сколько раз был скачан файл.

Свойства категории

Родительская категория

Категория в которую будет помещена данная.

Название

Название категории.

Описание

Краткое описание категории, отображаемое в списке категорий под ее названием (форматированный текст).

Свойства файла

Родительская категория

Категория, в которую будет помещен данный файл.

Название

Название файла.

URL

В это поле попадает URL файла после выбора его с помощь файлового менеджера (см. ниже). При необходимости можно вписать URL вручную.

Выбрать файл

Кнопка, при нажатии на которую открывается окно файлового менеджера, в котором можно выбрать нужный файл на сервере.

Подсказка

Если ваши файлы находятся на вашем локальном компьютере, а не на сервере, воспользуйтесь меню Действия → Загрузить файлы в файловом менеджере для загрузки их на сервер.
