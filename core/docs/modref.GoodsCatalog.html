Этот модуль позволяет организовать на сайте каталог-витрину товаров. Товары группируются по категориям, которые могут быть вложенными. Покупатель последовательно выбирая категории и подкатегории получает в итоге список товаров с кратким описанием. Щелкнув по названию товара покупатель попалает на страницу подробного описания, откуда он может произвести заказ. Заказ производится путем заполнения и отправки формы с полями предусмотренными настройкой профайла пользователя.

Подсказка

Для реализации более сложной схемы интернет-торговли с задействованием товарной корзины используется модуль интернет-магазина.

Общие действия для каталога

Действия → Новая категория

Создание новой категории с указанием ее свойств.

Действия → Новый товар

Добавление нового товара в каталог с указанием его свойств.

Действия → Сортировать документы в категории

Задание явного порядка сортировки товаров в данной категории. Товары с ненулевым приоритетом помещаются в начало и выстраиваются по его возрастанию, товары с нулевым приоритетом сортируются по полю, указанному в параметрах каталога.

Действия → Удалить группу документов

Удаление выбранных товаров в текущей категории.

Настройки → Настройки
Редактирование параметров каталога.
Настройки → Настройка навигации каталога
Редактирование параметров навигации каталога.
