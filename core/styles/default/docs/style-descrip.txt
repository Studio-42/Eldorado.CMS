Структура шаблонов ELDORADO.CMS:

- normal.html  - базовый шаблон страницы
- popup.html   - шаблон страниц, открываемых  в новом окне
+ alternatives - альтернативные шаблоны основной страницы
+ common       - шаблоны элементов, используемых на всех страницах (авторизация, поиск и тд)
	+ authFormsAndInfo - шаблоны авторизации и информации об авторизованном пользователе
		- authFormLeft.html - форма авторизации слева/справа
		- authFormTop.html  - форма авторизации сверху
		- userInfoLeft.html - инф. об авторизованном пользователе слева/справа
		- userInfoTop.html  - инф. об авторизованном пользователе сверху
	+ iCart - Шаблоны корзины покупателя
		- left.html - корзина слева/справа
		- top.html  - корзина сверху
	+ search - шаблоны форм поиска и страницы рез-тов поиска
		- left - форма поиска слева/справа
		- top -  форма поиска сверху
		- result.html - вывод результатов поиска
	- defaultTable.html - шаблон по умолчанию для 2-х колоночной таблицы (используется для формирования письма с заказом в инет-магазине и тд)
	- linked-objs.html  - списки связанных объектов в каталогах
	- messages.html     - инф сообщения и сообщения об ошибках
	- pager.html        - ссылки на следующие/предыдущие страницы для каталогов и новостей
	- pageTitle.html    - заголовок текущей страницы ("хлебные крошки")
	- popUpTitle.html   - тоже для нового окна 
+ css  - файлы css
	+ modules    - css файлы модулей
		- Container.css         - Стили модуля "Контейнер"
		- DocsCatalog.css       - Стили модуля "Каталог документов" 
		- EventSchedule.css     - Стили модуля "Расписание событий"
		- FAQ.css               - Стили модуля "Часто задаваемые вопросы"
		- FileArchive.css       - Стили модуля "Файловый архив"
		- Forum.css             - Стили модуля "Форум"
		- Glossary.css          - Стили модуля "Словарь терминов"
		- GoodsCatalog.css      - Стили модуля "Каталог товаров"
		- ImageGalleries.css    - Стили модуля "Альбомы изображений"
		- IShop.css             - Стили модуля "Интернет магазин"
		- linksCatalog.css      - Стили модуля "Каталог ссылок"
		- NavigationControl.css - Стили модуля "Управление навигацией"
		- News.css              - Стили модуля "Новости"
		- Poll.css              - Стили модуля "Голосования"
		- SiteMap.css           - Стили модуля "Карта сайта"
		- TechShop.css          - Стили модуля "Тех магазин"
		- UpdateClient.css      - Стили модуля "Обновление системы"
		- VacancyCatalog.css    - Стили модуля "Каталог вакансий"
	+ plugins    - css файлы дополнительных модулей
		- Calculator.css  - Стили доп модуля "Калькулятор"
		- InfoBlock.css   - Стили доп модуля "Инфо-блоки"
		- NewsTopics.css  - Стили доп модуля "Заголовки новостей"
		- Poll.css        - Стили доп модуля "Голосование"
		-RandomImage.css  - Стили доп модуля "Случайная картинка"
	+ services   - css файлы сервисов
		- iCart.css - Стили сервиса "Оформление заказа"
	+ ui-themes  - css файлы jquery-ui
	- contextmenu.css    - стили контекстного меню (файловый менеджер)
	- el-menu-float.css  - стили плавающего меню администратора
	- elborderselect.css - стили виджета выбора стиля бордюра (редактор)
	- eldialogform.css   - стили диалогового окна
	- elfinder.css       - стили файлового менеджера
	- elmenu.css         - стили выпадающего меню с js 
	- elpaddinginput.css - стили виджета выбора отступов (редактор)
	- elrte-inner.css    - стили для контента редактора
	- elselect.css       - стили виджета elselect (редактор)
	- fancybox.css       - стили fancybox (галерея изображений)
	- ie.css             - стили для IE
	- layout.css         - разметка страницы
	- normal.css         - базовый файл стилей. В нем переопределяются стили из styling.css & layout.css
	- print.css          - стили для media=print
	- styling.css        - сброс стилей и установка стилей для элементов (заголовки, параграфы и тд)
+ forms        - шаблоны форм
	- captcha.gdf      - шрифт для капчи
	- captcha.png      - фон для капчи
	- grid_form.html   - многоколоночная форма
	- simple_form.html - 2х колоночная форма
+ images       - папка для картинок
+ menus        - шаблоны различных видов основного и дополнительных меню
	- addMenuBottom.html     - шаблон нижнего доп меню 
	- addMenuTop.html        - шаблон верхнего доп меню
	- catMenuLeft.html       - шаблон меню каталогов слева
	- catMenuRight           - шаблон меню каталогов справа
	- catMenuTop.html        - шаблон меню каталогов сверху
	- mainMenuTop.html       - шаблон главного меню сверху 
	- menuLeft.html          - шаблон меню слева
	- menuRight.html         - шаблон меню справа
	- subMenuTop.html        - шаблон меню 2-го уровня сверху
	- techShopMNavLeft.html  - шаблон меню производителей из тех магазина слева
	- techShopMNavRight.html - шаблон меню производителей из тех магазина справа
	- techShopMNavTop.html   - шаблон меню производителей из тех магазина сверху
+ modules
	+ Container - шаблоны модуля "Контейнер"
		- default.html - список вложенных страниц в одну и 2 колонки
	+ DocsCatalog - шаблоны модуля "Каталог документов"
		- default.html - категории и документы в одну и 2 колонки
		- item.html    - вид одного документа
	+ EventSchedule - шаблоны модуля "Расписание событий"
		- default.html - список событий
		- event.html   - подробная информация о событии
	+ FAQ - шаблоны модуля "Часто задаваемые вопросы"
		- default.html - список вопросов и ответов
	+ FileArchive - шаблоны модуля "Файловый архив"
		- default.html - категории и файлы в одну и 2 колонки
	+ Finder - шаблоны модуля "Фаловый менеджер"
		- default.html - общий шаблон файлового менеджера
		- icons.html   - вид текущей директории в виде иконок
		- list.html    - вид текущей директории в виде списка
		- tree.html    - дерево директорий слева
	+ Forum - шаблоны модуля "Форум"
		- default.html - общий шаблон форумов
		- forums.html  - список форумов
		- post.html    - превью одного поста
		- profile.html - карточка пользователя
		- topic.html   - одна тема
		- topics.html  - все темы в форуме
		- users.html   - список пользователей
	+ GAStat - шаблоны модуля "Статистика"
		- browser.html   - браузеры
		- chartLine.html 
		- country.html   - страны
		- dashboard.html - сводная статистика
		- default.html   - общий шаблон
		- exit.html      - точки выхода
		- keyword.html   - ключевые слова
		- landing.html   - точки входа
		- medium.html    
		- os.html        - операционные системы
		- pagepath.html  - пути
		- pageviews.html - страницы
		- report.html
		- source.html
		- visits.html   - визиты
	+ Glossary - шаблоны модуля "Словарь терминов"
		- default.html - список терминов и букв
	+ GoodsCatalog - шаблоны модуля "Каталог товаров"
		- default.html - категории и товары в одну и 2 колонки
		- item.html    - вид одного товара
	+ ImageGalleries - шаблоны модуля "Альбомы изображений"
		- default.html - список альбомов 
		- gallery.html - один альбом
		- image.html   - изображение в новом окне
	+ IShop - шаблоны модуля "Интернет магазин"
		- default.html        - категории и товары в одну и 2 колонки
		- item.html           - карточка товара
		- mnfs.html           - список производителей
		- search-conf.html    - настройка поиска
		- search-form.html    - форма поиска над списком товаров
		- searchGridForm.html - табличная форма в поиске
		- types.html          - список типов товаров
	+ LinksCatalog - шаблоны модуля "Каталог ссылок"
		- default.html        - категории и ссылки в одну и 2 колонки
	+ NavigationControl - шаблоны модуля "Управление навигацией"
		- add-menu.html - настройка доп меню
		- default.html  - структура сайта
		- meta.html     - мета теги
		- modules.html  - список модулей
	+ News - шаблоны модуля "Новости"
		- default.html - список новостей в одну и 2-е колонки
		- news.html    - полное текст новости
	+ PluginsControl - шаблоны модуля "Управление доп модулями"
		- default.html - список доп модулей
	+ Poll - шаблоны модуля "Госования"
		- default.html - список голосований
	+ SimplePage - шаблоны модуля "Одиночная страница"
		- default.html - шаблон одиночной страницы (не обязательно)
	+ SiteBackup - шаблоны модуля "Резервные копии"
		- default.html - список резервных копий
	+ SiteControl - шаблоны модуля "Настройки сайта"
		- default.html - шаблон для всех табов кроме email
		- emails.html  - шаблон для настройки email
	+ SiteMap - шаблоны модуля "Карта сайта"
		- default.html - древовидный список страниц на сайте
	+ SitemapGenerator - шаблоны модуля "Генератор XML-карты сайта"
		- default.html - инф о XML-карте сайта
	+ TechShop - шаблоны модуля "Тех магазин"
		- compare.html	 - таблица сравнения товаров
		- default.html   - категории и товары в одну и 2 колонки
		- ft-groups.html - список тех характеристик
		- item.html      - карточка товара
		- manufact.html  - карточка производителя
		- manufacts.html - список производителей
	+ UpdateClient - шаблоны модуля "Обновление системы"
		- default.html   - инф о текущей и прошлых версиях
	+ UsersControl - шаблоны модуля "Управление пользователями"
		- default.html - список пользователей
		- groups.html  - список групп
		- profile.html - карточка пользователя
	+ VacancyCatalog - шаблоны модуля "Каталог вакансии"
		- default.html - категории и вакансии в одну и 2 колонки
		- item.html    - карточка вакансии
+ plugins      - шаблоны дополнительных модулей
	+ Calculator  - шаблоны доп модуля "Калькулятор"
		- adminCalculator.html - шаблон администратора (редактирование калькулятора)
		- adminList.html       - шаблон администратора (список, создание, удаление калькуляторов)
		- left.html            - калькулятор слева/справа
	+ InfoBlock   - шаблоны доп модуля "Инфо-блоки"
		- adminList.html - шаблон администратора (список инфо-блоков)
		- left.html      - шаблон инфо-блока слева
		- right.html     - шаблон инфо-блока справа
		- top.html       - шаблон инфо-блока сверху
	+ NewsTopics  - шаблоны доп модуля "Заголовки новостей"
		- left.html      - шаблон новостей слева
		- right.html     - шаблон новостей справа
		- top.html       - шаблон новостей сверху
	+ Poll        - шаблоны доп модуля "Голосования"
		- left.html      - шаблон голосований слева
		- right.html     - шаблон голосований справа
	+ RandomImage - шаблоны доп модуля "Случайная картинка"
		- left.html      - шаблон случайной картинки слева
		- right.html     - шаблон случайной картинки справа
		- top.html       - шаблон случайной картинки сверху
+ services     - шаблоны сервисов (оформление заказа)
	+ICart
		- default.html  - общий шаблон оформления заказа
		- delivery.html - шаблон выбора доставки
		- icart.html    - шаблон содержимого корзины
		- letter.html   - шаблон письма с заказом
		- summary.html  - шаблон сводной инф о заказе
