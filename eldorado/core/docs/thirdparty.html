Приложение A. Использованные сторонние продукты

В ELDORADO.CMS были использованы несколько сторонних разработок:

Текстовый редактор FCKEditor

Используется в качестве встроенного текстового редактора для редактирования текста. Сайт проекта: fckeditor.net.

Просмотрщик изображений LightBox

Используется для отображения рисунков в фотогалерее и в плагине случайного изображения. Домашняя страница проекта: http://www.huddletogether.com/projects/lightbox/
